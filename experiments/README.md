# Experiments Folder Notes

This folder contains the experiment log and test-specific artifacts.

## Canonical experiment docs
Use the Blank experiment template in `../templates/11-experiment-design.md` for any file that is intended to be a canonical experiment.

Minimum required sections for canonical experiments:
- Hypothesis
- Template refs
- What do I want to learn?
- Simplest pass/fail test
- Pass criteria
- Fail criteria
- Method
- Duration
- Cost
- Results
- Insight
- Action
- Template docs updated

## Non-canonical supporting files
Some files here are supporting materials rather than canonical experiments, for example:
- interview guides
- design frameworks
- synthesis docs
- matrices
- raw or semi-structured notes

Those do not need full template compliance, but they should be clearly named and should not pretend to be a canonical pass/fail experiment.
