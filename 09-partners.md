## Understand Potential Partner Types
> - **Technology partners:** Other agent frameworks, LLM providers, blockchain infrastructure
> - **Distribution partners:** Launchpads, developer platforms, agent marketplaces
> - **Integration partners:** Protocol teams who adopt Combinator, agent operators who join Hub
> - **Standards partners:** MCP ecosystem, agent protocol working groups

---

## List your target partners

### Necessary partners

**For Hub:**
> - Other agent runtime developers (CrewAI, LangGraph, AutoGPT) — for cross-runtime A2A messaging
> - OpenClaw core team — ActiveClaw is a fork, need to stay aligned or differentiate
> - MCP ecosystem — potential integration path for agent discovery

**For Combinator:**
> - Token launchpad platforms — for embedded futarchy governance at launch
> - Existing DAOs looking for governance solutions
> - Market makers / liquidity providers for decision markets

**For ActiveClaw:**
> - LLM providers (Anthropic, OpenAI, Google) — API access and pricing
> - Tool/skill developers — expanding agent capabilities

### What they will provide
> - Cross-runtime agents joining Hub (network effects)
> - Distribution via launchpad integrations (Combinator)
> - Liquidity for decision markets (Combinator)
> - Developer tools and integrations (ActiveClaw ecosystem)

### How the Company will reciprocate
> - Open-source infrastructure that partners can build on
> - Hub as shared coordination layer (not locked to our runtime)
> - Revenue share or co-marketing for launchpad partnerships
> - Integration support and documentation
