## Dependency Analysis

### Workflow/lifestyle changes on the part of the customer?
> Customers / operators may need to permit denser coordination, lower human intervention, and some security tradeoffs to unlock much higher throughput.

### Other products required?
> - model access / API access
> - wallets / chain access where crypto components matter
> - long-running agent workflows that justify coordination overhead

### Economic conditions?
> - crypto market conditions affect speculative activity and Combinator relevance
> - AI market growth affects urgency around long-running agent systems
> - cost of models affects how many long-running agents people are willing to run

### Behavior changes?
> - operators need to treat agents as network participants, not only isolated tools
> - teams need to invest in routing / coordination standards
> - customers may need to accept less isolation and more interaction density to get the throughput gains they want
